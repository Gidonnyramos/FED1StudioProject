Web Sandbox: Intro to HTML

Front-end Development

Front-end development is the tier of full-stack deveopment that deals with the visual elements of an application with which a user interacts. It consists of HTML, CSS, and JavaScript as these are the only languages that a browser can understand.
If you were to compare this tier of development to a human body, it would be like the skin.

HTML

Hypertext Markup Language is used to define the content and structure of a web page. It is the most important of the front-end languages because it provides content to the user and gives CSS and JS something to target.

CSS

Cascading Style Sheets are used for the formatting and layout of a web page. It was introduced as a way to separate the concerns of markup and formatting into different files.

JavaScript

JavaScript allows us to make web pages more interactive by performing programming logic in the front end, reacting to page events, and validating form data. It is one of the most prominent development languages in the world.

Lorem Ipsum

Lorem Ipsum is sample text. It is used in the mockup phase of page development to lay out text on the page in order to see how it will format. It comes from the print industry as a way of laying out text content that is nonsensical so that designers can focus on formatting rather than content and so that we don't have to have the final copy of the page content.
